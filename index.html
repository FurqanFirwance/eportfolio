Lab Submissions

Lab 1: Basic Layout with Jetpack Compose

Reflection:

This lab introduced me to the declarative nature of Jetpack Compose. I learned how to structure UI using Column, Row, and Box, realizing how much cleaner it is compared to the traditional XML layouts.

Lab 2: Interaction

Reflection:

Learned the power of State management. Building interactive elements showed me how Compose updates the UI automatically when data changes—a massive improvement for user experience.

Lab 3: Material Design

Reflection:

Implementing Material Design 3 made the app feel professional instantly. Using themes, typography, and elevation helped me understand how to build visually consistent applications.

Lab 4: Navigation

Reflection:

Navigation was the key to unlocking complex app flows. I learned how to pass arguments between screens and manage the back stack effectively using NavHost.

Lab 5: Room (Database)

Reflection:

Room taught me about data persistence. Storing user logs locally is crucial for offline availability, and working with DAOs and Entities sharpened my backend logic.
