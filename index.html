Laboratory Work

Lab 1: Basic Layout with Jetpack Compose

Reflection: This lab was my gateway into declarative UI design. Moving from XML to Jetpack Compose forced me to rethink how layouts are constructed. I learned the power of composable functions and how creating modular UI components leads to cleaner, more maintainable code compared to the traditional hierarchical XML approach.

Lab 2: Interaction

Reflection: I focused on state management in this lab. Understanding the difference between stateful and stateless composables was crucial. By managing user inputs and trigger events, I learned how Compose automatically recomposes the UI, ensuring that the app remains synchronized with the underlying data model.

Lab 3: Material Design

Reflection: Implementing Material Design 3 taught me that a great app is not just functional but also intuitive. I focused on using consistent typography, color schemes, and elevation. This experience deepened my appreciation for design language as a tool to improve user retention and accessibility.

Lab 4: Navigation

Reflection: Navigating through multiple screens taught me the importance of the NavHost and backstack management. Passing data between screens was a significant challenge that required a solid grasp of argument passing and clean architectural separation, ensuring the app feels like a unified experience.

Lab 5: Room (Database)

Reflection: Persistence was the missing piece of my app's architecture. By integrating the Room Database, I learned to abstract data access through DAOs and Entities. The ability to cache data locally transformed my app from a simple interface into a robust, offline-capable tool, providing a seamless user experience.
