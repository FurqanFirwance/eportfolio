Laboratory Work

Lab 1: Basic Layout with Jetpack Compose

Reflection: This lab served as my foundational step into declarative UI development. I learned to leverage Composable functions to replace legacy XML, realizing the efficiency and speed of modern Android UI creation.

Lab 2: Interaction

Reflection: By implementing state-based interactions, I grasped how UI components should react to user events in real-time. This solidified my understanding of state hoisting in Jetpack Compose.

Lab 3: Material Design

Reflection: Material Design 3 implementation taught me the importance of visual hierarchy and theme consistency. It transformed my basic layouts into professional, industry-standard interfaces.

Lab 4: Navigation

Reflection: Navigating between screens highlighted the importance of a clean architecture. Managing the backstack and passing parameters correctly was crucial for building a cohesive multi-screen application.

Lab 5: Room(Database)

Reflection: Integrating Room was a breakthrough in data management. I learned how to define Entities and DAOs to persist user data locally, ensuring my application remains reliable even without internet access.
